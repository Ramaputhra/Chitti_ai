# CHITTI Desktop Companion MVP - Roadmap

## Objective
Deliver a stable, local-first, privacy-respecting Desktop AI Companion based on the frozen CHITTI Project Blueprint v1.0.

## Core Tenets
1. **Desktop Focused:** Prioritize desktop integration, UI automation, and seamless local execution.
2. **Local First:** Run all foundational inference locally without mandatory cloud connections.
3. **Event-Driven:** Utilize the Core Event Bus for decoupled, robust communication.
4. **Experience-Driven Development:** Deliver complete user flows (e.g. Wake -> Converse -> Automate).

---

## Roadmap Phases

### Phase 1: MVP Alpha (Sprint 57)
- Stable End-to-End Execution Loop
- Local LLM Integration (Ollama)
- Basic Memory and Knowledge Retrieval
- Deterministic Task Execution and Desktop Automation (Capabilities)

### Phase 2: MVP Beta (Sprint 59)
- Refined Context Synthesis and Local Screen Understanding
- Polish Desktop App (Tray, Notifications, Overlay)
- Performance and Resource Monitoring

### Phase 3: MVP v1.0 Release (Sprint 60+)
- Complete System Packaging
- User Profiles and Stable Persistence
- Security and OTA Update Frameworks

## Future (MVP+)
*These features are frozen in the repository and will be activated post-v1.0:*
- Abstract Cognitive Workflows and Goal Runtimes
- Advanced Knowledge Graph and Identity Resolution
- Adaptive Intelligence and Prediction
- Robotics and Cloud Extensions (Archived for future phases)
