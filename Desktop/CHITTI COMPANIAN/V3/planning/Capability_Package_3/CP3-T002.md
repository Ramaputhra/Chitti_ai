# Task ID: CP3-T002

**Title:** Environmental & Internal Providers
**Owner:** ESP32 Firmware Engineer / Hardware API Engineer
**Priority:** High

## Description
Implement the `IRobotProvider` interface. Implement the core sensors: `DistanceProvider`, `IMUProvider`, `TouchProvider`, and `BatteryProvider`. Crucially, these providers must not dump raw numbers (e.g., raw accelerometer vectors or "pin 4 active") directly to the Planner. They must translate telemetry into semantic states (`MotionState.Fallen`, `TouchState.Petting`, `EnvironmentState.Approaching`) within the `RobotModel`.
