# Sprint 11: Knowledge Graph Runtime

## 1. Goal
Establish the Knowledge Graph Runtime to store facts. Introduce the `KnowledgeRepository`, `KnowledgeEdge`, and split Search and Retrieval into highly structured pipelines.

## 2. Deliverables
- `KnowledgeRepository`, `KnowledgeManager`, `KnowledgeSnapshot`
- `ArtifactStore` (Immutable), `RelationshipStore`, `RelationshipRegistry`
- `SearchEngine` (Keyword, Graph, Semantic modes)
- `RetrievalEngine` and `KnowledgeContextBuilder`
- Platform Explorer Update
