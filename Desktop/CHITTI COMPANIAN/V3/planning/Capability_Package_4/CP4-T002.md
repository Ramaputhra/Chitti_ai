# Task ID: CP4-T002

**Title:** Emotion & Animation Composition
**Owner:** Avatar Engineer / Emotion Engineer
**Priority:** High

## Description
Implement the `EmotionMapper` and `AnimationComposer`. The Mapper isolates the Planner from servo-level details, converting an abstract intent (e.g., `JOY`) into concrete targets (Eyes: Bright, LED: Blue). The Composer takes those targets and builds a precise millisecond-timeline (`ExpressionSequence`) representing the full behavioral arc (e.g., 0ms: Eyes Open -> 150ms: Smile -> 1000ms: Idle).
