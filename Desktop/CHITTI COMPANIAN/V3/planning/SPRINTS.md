# CHITTI Sprint Roadmap

The following is the strict execution sequence for the CHITTI Companion project, re-optimized to reach Milestone 1 ("Hey Chitti" vertical slice) as early as possible.

- **Sprint 1**: Foundation Complete (Architecture, Config, Logging, DI, Events)
- **Sprint 2**: Runtime Complete (Settings, Storage, State, Scheduler, Jobs, Service, Resource)
- **Sprint 3**: Audio Runtime (Mic, Speaker, VAD, Streaming Pipeline)
- **Sprint 4**: Language Runtime (Engine, Context, Intent, Planner, Skill Dispatcher, Response)
- **Sprint 5**: Capability Integration (Scenario Runner, Regression, Skill Registry, App Boot)
- **Sprint 6**: Conversation Runtime (Audio State, Wake Word, Sessions, Interrupts)
- **Sprint 7**: Real Speech Providers (Whisper, Edge TTS)
- **Sprint 8**  | **Intelligence Runtime** | Context Builder, Prompt Builder, LLM Router, Safety Layer, Tool Manager |
| **Sprint 9**  | **Desktop Runtime**      | OS abstraction, Hotkeys, Tray, Notifications |
| **Sprint 10** | **Desktop Tools**        | Read Mail, Open App, Search Files, Clipboard |
| **Sprint 11** | **Memory Runtime**       | Episodic, Semantic, Short/Long-term storage |
| **Sprint 12** | **Vision Runtime**       | Camera stream, OCR, Face Detection |
| **Sprint 13** | **Emotion Runtime**      | Emotion models, Mood transitions |
| **Sprint 14** | **Hardware Runtime**     | UART/BLE, Sensors, Servo integration |
