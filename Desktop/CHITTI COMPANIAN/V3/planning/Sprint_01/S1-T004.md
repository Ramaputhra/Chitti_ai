## Description
Implement `ILoggingService` and `LoggingService` using Loguru.
Supports structured logging, split files (`application.log`, `errors.log`, `performance.log`, `events.log`, `debug.log`), log rotation, and compression.
Includes a `@measure_time` decorator for performance logging and a `LogEvents` constants class.
