# Task ID: CP1-T002

**Title:** Image Preprocessor & Coordinate Mapper
**Owner:** Vision Engineer
**Priority:** High

## Description
Implement the `ImagePreprocessor` and `CoordinateMapper`. The preprocessor ensures we normalize, resize, and crop images *once* before passing them to the various Vision Providers. The coordinate mapper ensures that any bounding box produced by a provider is translated into a unified CHITTI Vision Coordinate system, allowing downstream robot targeting and attention framing.
