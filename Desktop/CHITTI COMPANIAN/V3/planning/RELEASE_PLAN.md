# CHITTI Product Release Roadmap

*Architecture is no longer the bottleneck. User experience is.*
From this point forward, every sprint answers one question: **What new thing can the user accomplish today that they couldn't yesterday?**

## Version 0.1 — Personal Assistant
**Goal:** Use CHITTI every day instead of manually checking these apps.
- Email triage
- Calendar assistant
- Meeting preparation
- Notes
- Browser research
- File organization

## Version 0.2 — Desktop Operator
**Goal:** Real desktop automation. *"Open VS Code, Obsidian, Chrome, and yesterday's screenplay."*
- Launch applications
- Control windows
- Clipboard intelligence
- Multi-step workflows
- Workspace restoration

## Version 0.3 — Creative Assistant
**Goal:** The user's specific workflow becomes a first-class capability.
- Screenplay brainstorming
- Character memory
- Scene continuation
- Story research
- Gemini image prompt generation
- Storyboard assistance

## Version 0.4 — Autonomous Agent
**Goal:** CHITTI starts working even when you don't explicitly ask.
- Monitor email
- Watch folders
- Scheduled reports
- Daily briefings
- Weekly summaries
- Automatic meeting preparation

## Version 1.0 — Desktop Companion
**Goal:** Flawless daily operation.
By this point, it should feel natural to say:
- *"Hey Chitti..."*
- *"Continue my work."*
- *"Prepare my meeting."*
- *"Summarize today's emails."*
- *"Find the latest version of my script."*
...without thinking about which application performs each task.

## Post 1.0 — Embodiment (The Robot Head)
**Goal:** The desktop application is already the daily assistant. The robot head simply becomes **another interface** to the exact same intelligence pipeline.
