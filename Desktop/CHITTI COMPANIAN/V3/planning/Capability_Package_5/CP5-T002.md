# Task ID: CP5-T002

**Title:** Desktop Context Runtime
**Owner:** Desktop Lead / Integration Engineer
**Priority:** High

## Description
Implement the `DesktopContextRuntime` to expose centralized OS-level metrics. It aggregates `Focused Window`, `Clipboard`, `Recent Files`, `Recent URLs`, `Notifications`, `Idle Time`, and `Power State`. This allows the Planner and productivity applications to easily append context to their actions without scattering OS-level API calls across the codebase.
