## Description
Implement the final missing cross-cutting platform service: the `DiagnosticsRuntime`. This serves as the platform's "black box recorder" before entering the wild world of third-party productivity integrations. It unifies the `HealthMonitor`, `StructuredLogger`, `DistributedTracer`, `MetricsCollector`, `PerformanceProfiler`, `CrashRecorder`, and critically, the `SessionReplay` module to enable deterministic debugging of multi-step AI workflows.
