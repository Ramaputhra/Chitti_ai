# CHITTI Vision Statement

**CHITTI is an AI Desktop Companion.**

It is not an AI framework. It is not a generalized operating system. It is a focused, capable product designed to assist, observe, present, and automate on behalf of the user.

## Core Directives

1. **Companion First:** CHITTI exists to augment the user's desktop experience. Every capability added must answer the question: *"What new thing can the user do after this change?"*
2. **Invisible Architecture:** CHITTI's cognitive abilities (Planning, Workflow Translation, Goal Assessment) operate invisibly beneath the surface. The architecture exists to serve the Companion, not the other way around.
3. **Presentation Matters:** CHITTI communicates using rich HTML, timelines, galleries, and expressions—not just raw text.
4. **Physical Embodiment:** CHITTI is built with the future in mind. The ESP32 robot and the desktop avatar are not separate AIs; they are simply different expressions of the same unified CHITTI brain.
