# CHITTI V2 — THEME SYSTEM SPECIFICATION

## 1. Supported Themes
- `Dark`: Fluent dark palette with glassmorphic opacity (`0.85`).
- `Light`: Fluent light palette with glassmorphic opacity (`0.90`).
- `System`: Follows host OS preferences.

## 2. Extensibility
Future themes register dynamically with `ThemeRegistry` without platform code changes.
