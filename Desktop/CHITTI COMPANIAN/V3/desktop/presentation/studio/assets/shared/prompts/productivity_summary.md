# Productivity Summary Prompt Template

## System Intent
Synthesize factual productivity session data into an executive summary.

## Data Inputs
- Session ID: `{{session_id}}`
- Primary Application: `{{top_application}}`
- Total Executions: `{{total_execution_count}}`
- Success Rate: `{{success_rate_percent}}%`

## Formatting Rules
1. Maintain strict factual precision.
2. Do not invent emotions or subjective intent.
3. Keep concise and clear.
