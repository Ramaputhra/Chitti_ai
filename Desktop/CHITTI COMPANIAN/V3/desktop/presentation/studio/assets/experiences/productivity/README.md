# Productivity Experience Documentation

## Purpose
Renders session telemetry, execution statistics, active window timelines, and factual insight cards into structured dashboard, voice, and avatar presentation artifacts.

## Bundle Consumed
- `AnalyticsPresentationBundle` (`BundleType.ANALYTICS`)

## Supported Renderers
- `DashboardRenderer` (`SupportedRenderer.DASHBOARD_RENDERER`)
- `VoiceRenderer` (`SupportedRenderer.VOICE_RENDERER`)
- `AvatarRenderer` (`SupportedRenderer.AVATAR_RENDERER`)

## Available Layouts
- `ExecutiveDashboard.tsx`
- `ModernDashboard.tsx`
- `GlassDashboard.tsx`
- `MinimalDashboard.tsx`

## Available Themes
- `dark.css`
- `light.css`
- `glass.css`
- `terminal.css`
- `accessibility.css`
