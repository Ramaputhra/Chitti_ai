# CHITTI V2 — IDENTITY AUTHORING GUIDE

Welcome to the **CHITTI V2 Identity Authoring Guide**. This guide explains how to edit biography, philosophy, creator profile, greetings, canonical responses, and capabilities.

---

## 1. Directory Structure (`desktop/character/identity/profiles/default/`)
- `identity.json`: Core profile fields (`display_name`, `creator`, `version`, `wake_names`).
- `biography.md`: Origin, evolution, purpose, design goals.
- `mission.md`: Mission statement and long-term vision.
- `philosophy.md`: Design principles and ethics.
- `beliefs.md`: Core behavioral values.
- `creator_profile.md`: Creator info (public information only).
- `speech_rules.md`: Greeting style, user nickname, forbidden phrases.
- `self_knowledge.md`: Capabilities and platform integrations.
- `boundaries.md`: What CHITTI admits, claims, and clarifies.
- `canonical_responses.md`: Authored responses to common identity queries ("Who are you?", "Who created you?", etc.).

---

## 2. How to Edit
1. Open the target Markdown document in `desktop/character/identity/profiles/default/`.
2. Edit the text.
3. Save the file. `IdentityRuntime` will automatically reload the document live!
