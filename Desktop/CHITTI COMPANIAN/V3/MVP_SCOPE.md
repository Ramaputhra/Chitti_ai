# MVP Scope Definition (Version 1.0)

This document strictly defines the features required for Version 1.0 (MVP Alpha/Beta). Everything outside this list is considered MVP+ and is currently frozen or archived.

## In Scope for MVP v1.0
1. **Local-First AI Execution**: Support for local LLMs (Ollama) as the primary intelligence engine.
2. **Local Voice Pipeline**: Offline Wake Word, Whisper STT, and Piper TTS.
3. **Core Desktop Automation**: Ability to launch basic OS applications (Notepad, Browser) and set timers.
4. **Basic UI Feedback**: A minimal system tray icon and visual overlay indicating state (Listening, Thinking, Speaking).
5. **Basic Memory**: Storage of recent conversational history and factual recall.
6. **Executable Packaging**: A bundled Windows installer/executable.

## Out of Scope for MVP v1.0 (Deferred to MVP+)
- Knowledge Graphs and Entity Resolution
- Autonomous web browsing and web scraping
- Advanced abstract multi-step cognitive workflows
- Vision/Camera processing (Facial recognition, emotion detection)
- Robotics integration
- Cloud syncing of memory profiles
